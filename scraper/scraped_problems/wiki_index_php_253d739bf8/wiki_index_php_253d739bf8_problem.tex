\subsection*{Problem}
The windshield wiper on the driver's side of a large bus is depicted below. What is the area of the windshield cleaned by the wiper, in square feet, to the nearest hundredth? (Assume that the windshield is a flat vertical surface.) $\textbf{(A) } 9.68 \qquad \textbf{(B) } 10.14 \qquad \textbf{(C) } 10.50 \qquad \textbf{(D) } 11.32 \qquad \textbf{(E) } 12.00$

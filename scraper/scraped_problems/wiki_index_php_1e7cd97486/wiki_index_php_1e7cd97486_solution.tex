\subsection*{Solution}
We are given $0.2(10) = 2$ pounds of cashews in the first box. Denote the pounds of nuts in the second nut mix as $x.$ \[5 + 0.2x = 0.4(10 + x)\] \[0.2x = 1\] \[x = 5\] Thus, we have $5$ pounds of the second mix. \[0.4(5) + 2 = 2 + 2 = \boxed{\text{(B) }4}\] ~pigwash ~yuvaG (Formatting) ~LucasW (Minor $\LaTeX$ ) ~Kirbyisdabestboiii (Very very very minor edits)

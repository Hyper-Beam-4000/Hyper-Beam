\section*{{2018 USAMO Problems/Problem 2}}
Source: AoPS Wiki

\subsection*{Problem}
Find all functions $f:(0,\infty) \to (0,\infty)$ such that \[f\left(x+\frac{1}{y}\right)+f\left(y+\frac{1}{z}\right) + f\left(z+\frac{1}{x}\right) = 1\] for all $x,y,z >0$ with $xyz =1.$

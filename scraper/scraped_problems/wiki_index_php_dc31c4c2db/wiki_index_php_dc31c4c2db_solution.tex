\subsection*{Solution}
We find that $i(i-3)=-1-3i$ and $(i-1)(i-2)=-1-3i+2=1-3i$ , so
\begin{align}
i(i-1)(i-2)(i-3)&=i(i-3)\cdot (i-1)(i-2)\\
&=(-1-3i)(1-3i)\\
&=-(1+3i)(1-3i)\\
&=-(1^2-(3i)^2)\\
&=-(1+9)\\
&=\boxed{\textbf{(D)}~-10 }
\end{align} ~ eevee9406
